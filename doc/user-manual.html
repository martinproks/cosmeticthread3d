Introduction

Cosmetic Thread 3D (nickname ct3d) is tool for visual simplicity of threads. Real 3D geometry of thread is very complex, consuming a lot of computer sources and for most (mechanical) designers just too complicated and meaningless. If the designer creates real thread geometry of a nut and a bolt, he needs to set correctly the orientation of the nut against the bolt to match thread surfaces. If the designer changes the clamping distance of the nut on the bolt, he needs to match the thread surfaces again. It is nightmare for normal designer and assembly.

Most time, designers needs to know and see (in 3D and in drawings) information like:

  • Thread major and minor diameter.
  • Thread length.

It could be good (but not necessary) to add some other information to the 3D model. Information that could be usable for drawings or CNC machining or FEM calculations.

1. Cosmetic Thread 3D types

FreeCAD has two major types of 3D geometry. Part geometry and PartDesign geometry. Each of them has different approach how to create model tree. It is necessary to handle each of them differently. For this reason, there are two types of ct3d and one transitional approach.

  • Part version of ct3d.
  • PartDesign version of ct3d.
  • Part version of ct3d applied to PartDesign base body.

2 Part version of ct3d

2.1 General description of Part version

Part version of ct3d is dedicated for geometry based on Part Workbench. The tools have extension P0 .. P4 in names. There are 5 visual types of the ct3d Part version tools - P0, P1, P2, P3 and P4. All of them works in the same way, they differ just in visual representation. You can test them and select/use the one You like the most. This manual describes the P0 type, but all of them have the same workflow and user interface.

The thread could be Internal or External. The tools are Internal and External too.

2.2 Internal ct3d - Part version - P0

  1. First You have to have Part geometry model with hole for thread. For example block with a hole.

  2. You should to activate Part with the object (if the object is inside Part).

  3. Choose Cosmetic Thread 3D workbench in the Workbench menu.

  4. A Cosmetic Thread 3D menu appears in the menu tool bar. Choose internal cosmetic thread P0.

  5. An attachment editor appears. Select hole starting edge (circle or arc) and mode Concentric.

    Note: The edge defines plane where the tread starts. It means if the hole is chamfered, the chamfer edge should be selected.

  6. Zoom the start of the hole. There is directional arrow (cone). The base of the cone represents starting plane of the thread and point of the cone shows the direction of the ct3d to be created.

    The cone is going into the hole - that is OK - in the image here. But if the cone pointing out of the hole, You should flip sides in the Attachment editor.

    Note: If the cone is too small or too big for the hole, try Apply button in the Attachment editor. The cone scale according to the selected circle.

  7. Press OK in the Attachment editor.

  8. And a thread User Interface (Thread UI) appears. You can select thread type (for example Metric Coarse thread ...) and thread dimension.

    The parameters You can edit are marked green on a picture below, parameters dependent on a specific thread are marked red and You can't edit them from Thread UI.

    Note: The thread dimension is estimated/guessed from circle/arc selected at attachment editor. It works well usually and estimates the dimension correctly, but sometimes it fails and set incorrect thread dimension. You should check the value and select the correct one.

    Note: You can press Apply to update geometry representation of the thread according to changes in parameters You made.

  9. Parameters Thread tolerances, Thread roughness and Length tolerances doesn't have impact to the geometry representation of the ct3d. You can left them empty or fill them as You wish. These parameteres are for future possibility to send information into TechDraw and CNC.

    Checkbox Thread throught hole/bolt is for termination annulus of thread. If You set it, the annulus is not shown.

    If You select Use Threads group, the thread will be created inside group Threads. If You unselect this option, no group will be created and the thread will be created in the Part object or on top of document.

  10. If You press OK, the cosmetic thread is created with parameters You set.

    You can edit all parameters freely in the Data window. Even the parameters that has not be possible to edit in the Thread UI.

2.3 External ct3d - Part version - P0

  1. First You have to have Part geometry model with cylinder or bolt-like geometry for thread. For example simple stud with a chamfer (the chamfer is not important, just example).

  2. You should to activate Part with the object (if the object is inside Part).

  3. Choose Cosmetic Thread 3D workbench in the Workbench menu.

  4. A Cosmetic Thread 3D menu appears in the menu tool bar. Choose external cosmetic thread P0.

  5. An attachment editor appears. Select thread starting edge (circle or arc) and mode Concentric.

    Note: The edge defines plane where the tread starts. It means if the rod/bolt/stud/whatever is chamfered, the chamfer edge should be selected, like here in this example.

  6. Zoom the start of the stud. There is directional arrow (cone). The base of the cone represents starting plane of the thread and point of the cone shows the direction of the ct3d to be created.

    If the cone pointing out of the stud (like here at this example), You should flip sides in the Attachment editor.

    Note: If the cone is too small or too big for the stud, try Apply button in the Attachment editor. The cone scale according to the selected circle.

  7. Press OK in the Attachment editor.

  8. And a thread User Interface (Thread UI) appears. You can select thread type (for example Metric Coarse thread ...) and thread dimension.

    The parameters You can edit are marked green on a picture below, parameters dependent on a specific thread are marked red and You can't edit them from Thread UI.

    Note: The thread dimension is estimated/guessed from circle/arc selected at attachment editor. It works well usually and estimates the dimension correctly, but sometimes it fails and set incorrect thread dimension. You should check the value and select the correct one.

    Note: You can press Apply to update geometry representation of the thread according to changes in parameters You made.

  9. Parameters Thread tolerances, Thread roughness and Length tolerances doesn't have impact to the geometry representation of the ct3d. You can left them empty or fill them as You wish. These parameteres are for future possibility to send information into TechDraw and CNC.

    Checkbox Thread throught hole/bolt is for termination annulus of thread. If You set it, the annulus is not shown. In this example, the thread is throught - it is going to the end of the cylinder and terminating out of material.

    If You select Use Threads group, the thread will be created inside group Threads. If You unselect this option, no group will be created and the thread will be created in the Part object or on top of document.

  10. If You press OK, the cosmetic thread is created with parameters You set.

    You can edit all parameters freely in the Data window. Even the parameters that has not be possible to edit in the Thread UI.
